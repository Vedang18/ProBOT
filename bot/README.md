# ProBOT - Core bot

This is the ProBOT core. This project defines the flow and logic of the bot.

Environment details:

1. Install [NodeJS](https://nodejs.org/en/download/)
2. Install dependencies

    ``` npm install ```

3. Run bot

    ``` npm start ```

Preffered IDE: [**VSCode**](https://code.visualstudio.com/)
